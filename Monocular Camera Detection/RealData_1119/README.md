> 上述现象产生的原因，分析如下：   
> **1.已选择时间戳最相近的相机和激光数据，帧率不匹配，无法做到同步；**   
> **2.相机和激光之间的坐标转换目前不明确，给的四元数对应的转换矩阵存在问题；**   
> **3.相机的畸变影响；**
