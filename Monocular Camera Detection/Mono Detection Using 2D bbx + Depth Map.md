# 1. 问题描述
以KITTI为例的3D感知开源数据集的打标类别一般为车、人，对于相关需求：障碍物(锥桶)已至后续可能的交通灯等其它目标物没有相应数据，因此调研相关技术路线以达成其它类别目标物3D感知的目标。

# 3. 方案概述
由于2D单目感知的高精度，深度预估通用模型的发展，以及部分类别物体数据的缺失，选用了2D+深度的单目感知方案。
